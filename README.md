# سياسة الخصوصية لتطبيق "واحة المرح"

تاريخ الإصدار: 2026

تم بناء تطبيق "واحة المرح" كتطبيق تعليمي وترفيهي آمن ومخصص للأطفال. هذه الصفحة تُعلم الزوار بسياساتنا المتعلقة بجمع واستخدام والإفصاح عن المعلومات الشخصية.

**1. جمع المعلومات واستخدامها**
نحن نولي أهمية قصوى لخصوصية الأطفال. تطبيق "واحة المرح" **لا يطلب، ولا يجمع، ولا يخزن، ولا يشارك** أي معلومات شخصية (مثل الأسماء، أرقام الهواتف، أو الموقع الجغرافي). التطبيق يعمل بشكل كامل دون الحاجة للاتصال بالإنترنت (Offline) ولا يطلب من المستخدم إنشاء أي حساب.

**2. صلاحيات التطبيق (Permissions)**
يطلب التطبيق صلاحية واحدة فقط وهي "الوصول إلى مساحة التخزين" (Storage). 
الهدف الوحيد من هذه الصلاحية هو: إنشاء مجلد محلي على هاتف المستخدم يسمى (KidPhotos) لنسخ وعرض الصور داخل بطاقة "معرض الصور" في التطبيق. لا يتم رفع، أو مشاركة، أو إرسال هذه الصور إلى أي خوادم خارجية إطلاقاً؛ فكل شيء يبقى داخل هاتف المستخدم فقط.

**3. الإعلانات وروابط الطرف الثالث**
تطبيق "واحة المرح" آمن تماماً، فهو خالٍ من أي إعلانات تجارية، ولا يحتوي على عمليات شراء داخل التطبيق (In-App Purchases)، ولا يضم أي أدوات تتبع (Trackers) أو روابط لصفحات خارجية قد تنقل الطفل خارج بيئة التطبيق.

**4. التغييرات على سياسة الخصوصية**
